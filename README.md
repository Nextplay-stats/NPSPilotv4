# Next Play Stats — MVP site

Static site to host Power BI Publish-to-web dashboards. Includes role tabs, embed placeholders, Firebase Auth skeleton, accessibility contrast toggle, and Netlify deployment readiness.

---

## Prerequisites

- Node.js (recommended) for local dev scripts
- Git for version control
- Netlify account for deployment (free tier)
- Firebase project (optional now; required when enabling auth)

---
